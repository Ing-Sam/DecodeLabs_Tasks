# TASK 4 – Power BI Dashboard

## Objective
The objective of this project was to create an interactive Power BI dashboard that transforms analyzed data into meaningful visual insights to support data-driven decision-making.

## Tools Used
Power BI


## Data Visualization Techniques
-Bar Charts
-Column Charts
-Line Charts
-Pie Charts
-Cards
-Tables and Matrix Visuals

## Key Skills Demonstrated
-Data Visualization
-Dashboard Design
-Report Development
-Business Intelligence
-Data Storytelling
-Power BI Modeling

The dashboard presents complex data in a clear and interactive format, enabling users to identify trends, compare performance, and gain actionable insights quickly.
