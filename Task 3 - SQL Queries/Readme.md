# TASK 3 – SQL Data Analysis

## Objective
The objective of this project was to use SQL queries to extract meaningful insights from the dataset by filtering, sorting, grouping, and aggregating data.

## Tools Used
-MySQL Workbench

## SQL Concepts Applied
-SELECT statements
-WHERE clause for filtering records
-ORDER BY for sorting data
-GROUP BY for summarizing data
-Aggregate functions:
 --COUNT()
 --SUM()
 --AVG()

## Analysis Performed
-Retrieved relevant records from the dataset
-Filtered data based on specified conditions
-Sorted records to identify top and bottom performers
-Grouped data into meaningful categories
-Calculated totals, averages, and counts to generate business insights

## Key Skills Demonstrated
-SQL Fundamentals
-Data Querying
-Data Filtering
-Data Grouping
-Data Aggregation
-Business Insight Generation

##Outcome
The SQL queries transformed raw data into actionable insights that can support business decision-making and serve as a foundation for visualization in Power BI.
