# TASK 1 – Data Cleaning in Excel

## Objective

The objective of this project was to clean a raw dataset and prepare it for analysis by improving its quality, consistency, and accuracy.

## Tools Used
Microsoft Excel

##Data Cleaning Tasks Performed
-Removed duplicate records
-Handled missing or blank values
-Standardized text formatting
-Corrected inconsistent entries
-Removed unnecessary columns
-Checked for data quality issues

## Outcome
The cleaned dataset is accurate, consistent, and ready for exploratory data analysis, SQL querying, and Power BI visualization.

##Skills Demonstrated
-Data Cleaning
-Data Preparation
-Data Validation
-Excel Functions
-Data Quality Management
