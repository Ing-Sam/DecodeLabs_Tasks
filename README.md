# DECODELABS VIRTUAL INTERNSHIP PROJECTS - SAMUEL MENSAH

## Overview

This repository contains a collection of projects completed as part of my Data Analytics internship with Decodelabs. The projects demonstrate my ability to work with data from cleaning and exploration through SQL analysis and dashboard creation.

## Tools Used
-Microsoft Excel
-MySQL
-Power BI
## Repository Structure
-Project 1 – Data Cleaning (Excel)

 Performed data cleaning to improve data quality by handling missing values, removing duplicates, correcting inconsistencies, and preparing the dataset for analysis.

-Project 2 – Exploratory Data Analysis (Excel)

 Explored the cleaned dataset using Excel functions, pivot tables, and charts to identify trends, patterns, and key insights.

-Project 3 – SQL Data Analysis

 Used SQL queries to extract insights from the dataset through filtering, grouping, sorting, and aggregation functions such as COUNT(), SUM(), and AVG().

-Project 4 – Power BI Dashboard

 Built an interactive dashboard to visualize key performance indicators and communicate insights using charts, cards, and slicers.
